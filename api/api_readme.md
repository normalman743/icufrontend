# 校园LLM系统 API 文档

## 概述
这是校园LLM系统的API文档。系统使用MySQL存储关系数据，Chroma存储向量数据，本地文件系统存储文件，并集成了LangChain进行RAG（检索增强生成）处理。

## API文档结构

为了更好地组织API文档，我们将其分成了多个模块文件：

1. [API 索引与概述](api_index.md) - 主索引页面，包含所有API模块的链接和概述

2. 各API模块：
   - [认证与用户管理](api_auth.md) - 用户注册、登录和个人资料管理
   - [学期与课程管理](api_semester_course.md) - 学期和课程相关操作
   - [文件夹与文件管理](api_folder_file.md) - 文件和文件夹相关操作
   - [聊天、消息与AI功能](api_chat_message_rag.md) - 聊天、消息和AI/RAG功能
   - [系统管理](api_admin.md) - 管理员专用接口

3. [数据库设计](database.md) - 详细的数据库表结构设计

## 核心功能亮点

- **自动标题生成** - 聊天创建无需标题，系统会根据首条消息自动生成
- **智能RAG检索** - 根据聊天类型自动从不同知识库检索信息
- **文件去重优化** - 使用SHA256哈希实现物理文件去重，节省存储空间
- **流式响应** - AI对话支持流式响应，提供更好的用户体验

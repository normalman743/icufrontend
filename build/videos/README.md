## 🎬 视频使用说明

### Hero Section (首页) 的动画序列：

1. **页面加载时**：
   - 播放 `p2b.mp4` (凤凰→球转换)
   - 然后播放 `ball生成完毕.mp4`
   - 最后进入待机状态

2. **待机状态**：
   - 循环播放 `ball缓慢运动.mp4`

3. **鼠标悬停**：
   - 切换到 `ball快速旋转.mp4`
   - 鼠标离开后恢复到缓慢运动

4. **特殊效果（未来可添加）**：
   - `b2icu.mp4` - 球变成ICU文字
   - `ball强烈闪光.mp4` - 强调效果
   - `ball旋转生成（出现光电）.mp4` - 加载效果

- 格式：JPG 或 PNG

## 💡 提示

- 视频会自动播放（muted状态）
- 移动设备上会使用 playsInline 属性
- 建议视频文件大小 < 5MB 以保证加载速度
- 如果视频太大，可以压缩或转换为 webm 格式
